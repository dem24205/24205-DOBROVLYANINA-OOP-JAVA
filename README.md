# Стековый калькулятор

Консольный калькулятор, использующий стековую архитектуру для вычислений. Поддерживает переменные, базовые арифметические операции и сохранение состояния.

## Команды калькулятора

- `DEFINE <name> <value>` - создает переменную с указанным именем и значением  
  *Пример: `DEFINE PI 3.14`*

- `PUSH <number>` или `PUSH <variable>` - помещает значение на вершину стека  
  *Примеры: `PUSH 5`, `PUSH r`*

- `POP` - удаляет верхний элемент из стека

- `+` - складывает два верхних элемента стека  
  *Состояние стека: `[a, b] → [a+b]`*

- `-` - вычитает верхний элемент из предыдущего  
  *Состояние стека: `[a, b] → [a-b]`*

- `*` - умножает два верхних элемента  
  *Состояние стека: `[a, b] → [a*b]`*

- `/` - делит предыдущий элемент на верхний  
  *Состояние стека: `[a, b] → [a/b]`*

- `PRINT` - выводит верхний элемент стека в консоль

- `SAVE` <filename.json> - сохраняет текущее состояние калькулятора в файл `filename.json`

- `EXIT` - завершает работу калькулятора. При чтении команд из файла команда необязательна - программа завершится после обработки всех команд


## Структура сохраняемых данных

```json
{
  "stack": [ 78.5, 0.0 ],        // Текущее состояние стека (верхний элемент последний)
  "variables": {
    "PI": 3.14,                  // Все определенные переменные
    "r": 5.0
  },
  "history": [                   // История выполненных команд
    "DEFINE r 5",
    "DEFINE PI 3.14", 
    "PUSH r",
    "PUSH r",
    "*",
    "PUSH PI",
    "*",
    "PRINT"
  ]
}
